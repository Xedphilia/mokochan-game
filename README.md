# モコちゃんアプリ 試作版 v19

GitHub Pages でそのまま表示できる単体HTML版です。

## 内容

- `index.html`：アプリ本体
- 動画・画像素材は `index.html` 内に埋め込み済み
- `assets` フォルダ不要

## GitHubで動かす方法

1. GitHubで新しいリポジトリを作成
2. このフォルダ内の `index.html` をアップロード
3. リポジトリの `Settings` を開く
4. `Pages` を開く
5. `Build and deployment` の `Source` を `Deploy from a branch` にする
6. `Branch` を `main`、フォルダを `/root` にして `Save`
7. 数分後に表示URLが発行されます

## 注意

この版は確認用の軽量単体版です。
動画は表示確認しやすいように圧縮されています。
本番用では、動画・画像を `assets` フォルダに分けて管理する構成に戻す方が適しています。

## 現在の主な実装

- リビング背景動画
- idle / おもちゃ遊び / 椅子遊び の状態切替
- タップ時のhappy動画
- ロード画面
- ロード画面のフェードイン・フェードアウト
- 下部メニュー
- 左上：満腹度
- 右上：保有pt
